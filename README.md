Quick Start
Run XDDiscordTool.exe as Administrator

Navigate using number inputs (1-30)

Follow on-screen prompts for each tool

Press any key to return to main menu

Select 30 to exit

❓ FAQ
Q: Do I need to run as Administrator?
A: While not strictly required, running as Administrator ensures all tools function properly, especially those that interact with system-level operations.

Q: Is this tool safe?
A: XDDiscordTool is open-source and contains no malware. However, using some tools may violate Discord's Terms of Service. Use responsibly.

Q: Why do some tools require a bot token?
A: Certain Discord API endpoints require bot authentication with specific permissions to function.

Q: The tool shows "Invalid token" - what's wrong?
A: Ensure you've copied the complete token including all three parts separated by dots. Tokens expire and can be invalidated.

Q: Can I contribute to development?
A: Absolutely! Fork the repository, make your changes, and submit a pull request.
